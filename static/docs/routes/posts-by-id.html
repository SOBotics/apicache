Posts by ID (/posts/{ids})

Aliases: /questions/{ids}, /answers/{ids}

Fetches a set of posts identified by a semicolon-delimited list of post IDs.

Parameters

Required:

  • key - a Stack Exchange API key
  • site - the API site parameter for a Stack Exchange site

Optional:

  • page - page number of results to fetch
  • pagesize - number of results to return in one page
  • max_age - maximum age in seconds of cached results you want returned

Examples

Requesting posts 123, 124, and 125 from Stack Overflow

GET /questions/123;124;125?key=IKesi3924bsnsoe9((&site=stackoverflow HTTP/1.1

Requesting the same posts, but with a maximum cache age of five minutes

GET /questions/123;124;125?key=IKesi3924bsnsoe9((&site=stackoverflow&max_age=300 HTTP/1.1
