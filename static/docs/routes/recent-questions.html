Recent Questions (/questions)

Fetches recently-posted questions. This route does not support paging.

Parameters

Required:

  • key - a Stack Exchange API key
  • site - the API site parameter for a Stack Exchange site

Optional:

  • pagesize - number of results to return in one page
  • max_age - maximum age in seconds of the recent questions list

Examples

Recent questions on Super User

GET /questions?key=IKesi3924bsnsoe9((&site=superuser HTTP/1.1
